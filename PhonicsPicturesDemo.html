Phonics Pictures Font

How to Use

This font creates pictorial representations for each letter of the alphabet. Each letter shows a shape or picture of a word that begins with that letter.

To use this font:

  1. Install the font by double-clicking the PhonicsPicturesShapes.ttf file
  2. Select "Install Font" in the Font Book app
  3. Use the font in any application that supports custom fonts
